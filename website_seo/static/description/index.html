Setup & Configure

Default Parameters

After the successful installation of the ODOO SEO-PRO module a POP-UP Wizard will Open,You will have to fill the Required Parameter.

Required Parameter

  • Attribute Separator.
  • Value Separator.
  • Meta Title Size.
  • Meta Desc. Size.
  • Meta Keyword Size.
  • Meta Keywords.
SEO For Product

This module provide the essentials support in managing the SEO for Products

First of all you will have to generate the SEO Meta -Info for you Product for this just go through Sale-->Website SEO

  • Select Product in the SEO For List.
  • Select product attribute for managing the meta Title,Description, Keywords .
  • Assign/Change the Multi Attribute Separator.
  • Assign/Change the Multi Value Separator.
  • Click on Generate Now.

Fields For Product SEO

  • Product Name.
  • Category Name.
  • Description.
  • Product Description.
  • EAN13 Bar-code.
  • Company Name.

After Clicking the generate button , you will see the message about the Updation, message also include number of affected product with this SEO generation/Updation.

You can also edit the order of attribute from seo attribute menu.

Cookie - Consent Manager Basic

This module allows your customers to give their consent to various cookies on your website. Without a consent the functionality is disabled and the corresponding cookies deleted as requested by GDPR. The basic version provides an indication of essential cookies that are necessary for the operation of the site and the possibility for your customers to opt-out of google analytics.

Info: Please also note this additional module which gives your customers the possibility to give their consent to activate the odoo livechat. You can download it for free in the odoo appstore.

Consent Manager Interface

The main interface gives your customers an overview of which personal information is processed. All non-essential settings can be approved or rejected here with a single click.

Consent Manager Settings

In the settings interface your customers can manage individual options.

Configuration

You can adjust the configuration in the configuration file (static/src/js/klaro.config.js). With the value "privacyPolicy" you configure the link to your privacy policy. You can also explicitly provide a path and a domain for a given cookie. This is necessary if google analytics set cookies for a path that is not "/" or a domain that is not the current domain. If you do not set these values properly, the cookie can't be deleted.

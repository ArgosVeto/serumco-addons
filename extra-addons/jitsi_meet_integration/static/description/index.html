Quick Start

Installation

There are no dependencies so you can simply install the module.

Configuration

if you would like to use your own jitsi server add a new System Parameter with key: jitsi.server and value: your.server.name (only hostname)

Usage

Create an appointment, enable the Jitsi remote meeting and add the participants. Your participants will receive an email containing a direct link for opening the video conference.
